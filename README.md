# Desktop-Assistent-Voice
The desktop assistant will open programs or specific website trough out "speech_recognition" library of Python. 

In case in problem with the modules not being installed just run the followings commands in the terminal according to the error message received:

pip install pyaudio
pip install SpeechRecognition
pip install pyttsx3
